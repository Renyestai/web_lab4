# Лабораторная работа №4

## Сборка и запуск

Для сборки и запуска проекта необходимо поочередно ввести команды:

- ```npm run install```
- ```npm run build```
- ```npm run preview```

> Внимание! Часть картинок загружаются только при использоании VPN

## Тестирование

Для запуска всех тестов необходимо ввести команду:

- ```npm run test```
